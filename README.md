# NumPy, Pandas, Matplotlib, and Seaborn Examples

A comprehensive collection of examples demonstrating the core functionality of NumPy, Pandas, Matplotlib, and Seaborn - the essential data science libraries for Python.

## Overview

This repository contains practical examples and demonstrations of:

- **NumPy**: Numerical computing with arrays and mathematical operations
- **Pandas**: Data manipulation and analysis with DataFrames
- **Matplotlib**: Data visualization and plotting
- **Seaborn**: Statistical data visualization built on Matplotlib

## Installation

### Prerequisites

- Python 3.7 or higher
- pip (Python package manager)

### Setup

1. Clone this repository:
```bash
git clone https://github.com/ShezzTRANS/-NumPy-Pandas-Matplotlib-and-Seaborn.git
cd -NumPy-Pandas-Matplotlib-and-Seaborn
```

2. Install required packages:
```bash
pip install -r requirements.txt
```

## Usage

Each library has its own example file that can be run independently:

### NumPy Examples
```bash
python numpy_examples.py
```

Demonstrates:
- Basic array operations
- Array attributes and methods
- Array creation methods (zeros, ones, identity, arange, linspace, random)
- Mathematical functions (sqrt, exp, log)
- Statistical operations (mean, median, std, var)
- Array indexing and slicing
- Array reshaping and transposing
- Linear algebra operations

### Pandas Examples
```bash
python pandas_examples.py
```

Demonstrates:
- Creating DataFrames
- DataFrame information and statistics
- Data selection and filtering
- Data manipulation (adding, modifying, dropping columns)
- Sorting operations
- GroupBy operations and aggregations
- Missing data handling
- Merge and join operations
- Time series operations
- Apply operations and custom functions

### Matplotlib Examples
```bash
python matplotlib_examples.py
```

Creates visualizations including:
- Line plots
- Scatter plots
- Bar charts
- Histograms
- Pie charts
- Subplots
- Heatmaps
- Box plots

All plots are saved as PNG files in the current directory.

### Seaborn Examples
```bash
python seaborn_examples.py
```

Creates statistical visualizations including:
- Distribution plots (histogram with KDE)
- Scatter plots with regression lines
- Box and violin plots
- Correlation heatmaps
- Pair plots
- Count plots
- Line plots
- Various categorical plots (bar, point, strip, swarm)

All plots are saved as PNG files in the current directory.

### Run All Examples
```bash
python run_all_examples.py
```

Runs all examples in sequence and generates all visualizations.

## Examples Overview

### NumPy
NumPy is the fundamental package for scientific computing in Python. It provides:
- A powerful N-dimensional array object
- Sophisticated broadcasting functions
- Tools for integrating C/C++ and Fortran code
- Useful linear algebra, Fourier transform, and random number capabilities

### Pandas
Pandas is a fast, powerful, flexible, and easy-to-use data analysis and manipulation tool. It provides:
- DataFrame and Series objects for data manipulation
- Tools for reading and writing data
- Data cleaning and preparation
- Data aggregation and transformation
- Time series functionality

### Matplotlib
Matplotlib is a comprehensive library for creating static, animated, and interactive visualizations. It offers:
- Publication-quality figures
- Interactive figures that can zoom, pan, update
- Customizable styles and colors
- Support for various output formats

### Seaborn
Seaborn is a Python data visualization library based on Matplotlib. It provides:
- High-level interface for attractive statistical graphics
- Built-in themes for styling
- Tools for choosing color palettes
- Functions for visualizing univariate and bivariate distributions
